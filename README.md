# davis: Data and Algorithm Visualizer

Visualizer for data structures and algorithms in C.

## Data structures

- Tuple (static array)
- Vector (dynamic array)
- List (single, double)
- Queue (FIFO)
- Stack (LIFO)

- Map (hash table)
- Set (red-black tree)
- Heap (priority queue)
- Binary search tree

## Algorithms

- Heapsort
- Mergesort
- Quicksort
- Introsort

## Visualization

- Lines
- Rectangles
- Circles / elipses
- Plots (scatterplot, function plot)
- Grids (matrix)
- Charts (bar chart, histogram)
- Graphs (trees, adjacency list/matrix, decision)
- Vector field
